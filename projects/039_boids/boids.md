# Boids

Implementation of coordinated animal motion such as bird flocks and fish schools.
Based on Craig Reynolds concept.

![alt text](images/boids2.gif "boids animation")

## How to run

```commandline
python run_boids.py
```

Requirements:

* Python 3.7
* Pyside2
